About MegaMart

MegaMart is a demo online shopping website built as a capstone project using only HTML, CSS and JavaScript.

It includes a home page, product listing, shopping cart with localStorage, contact form validation and a responsive layout inspired by modern e-commerce sites.

Use this project to practice:

  • Layout with Flexbox and CSS Grid
  • Dynamic DOM updates
  • State management using localStorage
  • Performance and responsive design
